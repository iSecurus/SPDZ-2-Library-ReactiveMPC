#### Running the example code

- Make sure that SPDZ and the python library ```pexpect``` are correctly installed.
- Now modify ```port```, ```hostname``` and other configurations as needed in the sourcecode of ```codemaker.py``` and ```codebreaker.py```.
- Last but not Least you have to run
   ```python3 codebreaker.py```
   and
   ```python3 codemaker.py```
   in a shell.
